# ConstantSEO

AI-powered SEO agency website by [Constant Labs](https://constantlabs.ai), Dubai.

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Framer Motion
- Bilingual (English / Arabic with RTL)

## Development

```sh
npm install
npm run dev
```

## Deployment

Hosted at [seo.constantlabs.ai](https://seo.constantlabs.ai)
